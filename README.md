# Fan Ticket Hosting

## Concept

* CreationPermit Signer - Account that sign the permit for creation, must be enrolled in the registry contract.
* Issue Operator  - Account that take cares the token's creation. Could be any wallet.
* Clearing Operator - Account that take cares all the token's transfer (Transfer / Mint). Could be any wallet.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

$ npm run test:e2e
